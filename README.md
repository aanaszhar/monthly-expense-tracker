# Personal Expense Tracker
A simple, responsive web application designed to help you log and monitor your monthly expenses. The project focuses on simplicity, clear data visualization, and an intuitive user experience.

## Features
* **Expense Management**: easily add, edit, and delete your daily expenses
* **Smart Filtering**: 
  * filter expenses by year (2025–2026)
  * filter by specific expense categories using a multi-select dropdown
  * Search expenses by title or notes/remarks
* **Data Visualization**:
  * **Monthly Bar Chart**: visualize spending trends across the year
  * **Interactive Pie Chart**: category-wise breakdown for any selected month
* **Budget Tracking**: set a "Max Monthly Expense" limit; the page will highlight months that exceed your budget in red
* **CSV Export**: Export your records to CSV
* **Light/Dark mode**: Toggle between Light and Dark modes

## tech used
* **techs**: HTML, CSS, javascript, bootstrap, jQuery, chart.js
