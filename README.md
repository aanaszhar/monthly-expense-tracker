# Personal Expense Tracker

A simple, responsive web application designed to help you log and monitor your monthly expenses. The project focuses on simplicity, clear data visualization, and an intuitive user experience.

## Features

* **Expense Management**: Easily add, edit, and delete your daily expenses.
* **Smart Filtering**: 
  * Filter expenses by year (2025–2026).
  * Filter by specific expense categories using a multi-select dropdown.
  * Search expenses by title or notes.
* **Data Visualization**:
  * **Monthly Bar Chart**: Visualize spending trends across the year.
  * **Interactive Pie Chart**: Get a category-wise breakdown for any selected month.
* **Budget Tracking**: Set a "Max Monthly Expense" limit; the app will highlight months that exceed your budget in red.
* **Data Persistence**: Uses LocalStorage to save your data directly in your browser.
* **Data Control**: Export your records to CSV or clear all data with one click.
* **Theme Switching**: Toggle between Light and Dark modes for comfortable viewing.

## Technologies Used

* **HTML5**: Structural markup.
* **CSS3**: Styling and responsive design.
* **Bootstrap 5**: Frontend framework for a polished, mobile-friendly interface.
* **JavaScript (ES6+)**: Application logic and data management.
* **jQuery**: For simplified DOM manipulation and event handling.
* **Chart.js**: For rendering the interactive bar and pie charts.
